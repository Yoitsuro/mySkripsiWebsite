# mySkripsiWebsite

## How to run the project

### in Linux / MacOs

- run the backend with uvicorn

1.  active the virtual env first with `source [local/share/pipx/venvs/ipykernel/bin/activate]` [env-path/bin/active]
2.  then go to terminal type `uvicorn main:app --reload` then enter

- run the frontend

1. go to frontend dirrectory
2. run the liveserver and go to the frontend directory
